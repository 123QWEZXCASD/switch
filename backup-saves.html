    Резервное копирование и восстановление сохранений с помощью Checkpoint

    Разделы

    • Создание резервных копий сохранений
    • Восстановление резервных копий сохранений
    Если вы попали в это руководство впервые и не знаете как прошивается Switch, рекомендуем начать знакомство с ним с первой страницы
    Если вам помогло или просто понравилось это руководство, то мы принимаем пожертвования.
    По вопросам, касающимся взлома и руководства, обращайтесь сюда
    Если приставка зависла или не подаёт признаков жизни, выключите её удерживанием кнопки (POWER) 30 секунд, затем включите и пробросьте пейлоад!

    Создание резервных копий сохранений

    1. Запустите Homebrew Launcher
    2. Запустите Checkpoint
    3. Выберите аккаунт для которого хотите создать резервную копию, с помощью клавиши (ZR) или (ZL)
    4. Выделите игры для сохранений которых хотите создать резервную копию, с помощью клавиши (Y)
      • Если вы хотите выбрать все, зажмите (Y)
    5. Нажмите на клавишу (L) для того, чтобы сделать резервные копии сохранений выбранных игр
    6. При необходимости повторите для каждого из аккаунтов
    7. Резервные копии сохранений находятся на карте памяти по пути switch/checkpoint/saves. Скопируйте папку saves в надёжное место

    Восстановление резервных копий сохранений

    1. Убедитесь, что резервные копии необходимых сохранений находится по пути switch/checkpoint/saves на вашей карте памяти
    2. Запустите Homebrew Launcher
    3. Запустите Checkpoint
    4. Выберите в списке игру для которой хотите восстановить резервную копию сохранения и нажмите (A)
      • Если вы не видите нужную игру в списке, запустите её и поиграйте немного, чтобы консоль создала все необходимые записи в системе. Как правило достаточно просто запустить игру
    5. Выберите нужное сохранение из списка и нажмите на нём (A), нажмите (A), чтобы подтвердить восстановление
    6. Повторите для каждой игры, для которой хотите восстановить резервную копию сохранения
