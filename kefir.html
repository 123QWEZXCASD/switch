    KEFIR 424

    В контексте этого руководства - сборник, состоящий из выбранного кастома, необходимых программ и скриптов, которые все это установят правильным образом. Подробнее про кефир здесь. Зачем он нужен? Всё очень просто. Кефир содержит в себе самое полезное и проверенное временем ПО, которое в свежем кефире всегда идёт актуальной версии. Кефир устанавливается с помощью скрипта, который написан таким образом, чтобы удалять при установке все проблемные модули, из-за которых система моет работать нестабильно или не работать вовсе. При возникновении проблем, достаточно назвать версию кефира, чтобы сразу стало ясно какое именно ПО и какой версии сейчас на вашей приставке.

    ВНИМАНИЕ!

    • Перезагрузка в hekate происходит прямо из HOS через обычное меню перезагрузки. Просто зажмите (VOL-) во время сплешскрина кефира
    • В hekate можно вынуть карту из приставки, а затем, вставив обратно, запустить прошивку через hekate -> Launch -> Atmosphere
    • На новых версиях Atmosphere изменился способ входа в Homebrew Menu. Подробнее
    • Установка и обновление kefir проводятся одинаково!

    Инструкция по установке kefir

    Инструкции по автоматической установке (только для Windows)
    1. Скачайте свежий kefir Сборник, состоящий из выбранного кастома, необходимых программ и скриптов, которые все это установят правильным образом. Состав кефира и инструкцию по установке можно посмотреть в его репозитории (файл _kefir.7z)
    2. Распакуйте _kefir.7z в любое удобное место на ПК
    3. Вставьте в ПК карту памяти приставки
    4. Запустите install.bat из папки, в которую вы распаковали _kefir.7z
    5. Вводите ответы, соответствующие вашей ситуации
      • Если вы используете Caffeine, не забудьте выбрать его в опциях установщика
    6. Дождитесь окончания копирования
    7. Продолжайте следовать руководству
    Инструкция по установке вручную (все ОС)

    Вы можете получить доступ к карте памяти двумя способами:

    1. Достав карту памяти из приставки и вставив её в ПК
    2. Перезагрузите приставку через меню, вызываемое удерживанием кнопки (POWER)
    3. На сплеш-скрине кефира нажмите кнопку понижения громкости, чтобы попасть в hekate
    4. Находясь в hekate вы модете достать карту памяти и вставить её в ПК
    5. С помощью DBI
    6. Запустите DBI
    7. Подключите консоль к ПК по USB-кабелю
    8. В DBI выберите пункт “Run MTP Responder”
    9. На ПК смонтируется новое устройство - Switch, папка “External SD card” и есть ваша карта памяти * Помните, что вы не сможете установить kefir автоматически по MTP!

    Удаление файлов старых прошивок

    1. Если вы использовали пользовательские модификации, читы или переводы, переместите папку titles из папки sxos или ReiNX или atmosphere (название папки зависит от вашей прошивки) в корень карты памяти. Если не использовали, пропустите эту часть
    2. Удалите из папки ‘titles’ папки со следующими названиями (если они там есть):
      • 010000000000100D
      • 0100000000000032
      • 0100000000000034
      • 0100000000000036
      • 010000000000000D
      • 420000000000000E
      • 0100000000001000
      • 0100000000000352
      • 4200000000000010
    3. Удалите из корня вашей карты памяти все папки, кроме папок titles, sxos, emuMMC и Nintendo, если таковые там есть

    Для установки Atmosphere

    1. Скопируйте в корень карты памяти приставки содержимое архива atmo.zip из репозитория kefir Сборник, состоящий из выбранного кастома, необходимых программ и скриптов, которые все это установят правильным образом. Состав кефира и инструкцию по установке можно посмотреть в его репозитории
    2. Переместите папку titles из корня карты памяти в папку atmosphere
    3. Переименуйте папку titles в contents
      • Саму папку, а не её содержимое
    4. Исправьте атрибуты по инструкции ниже
    5. enjoy

    Для установки SX OS

    1. Скопируйте в корень карты памяти приставки содержимое архива sxos.zip репозитория kefir Сборник, состоящий из выбранного кастома, необходимых программ и скриптов, которые все это установят правильным образом. Состав кефира и инструкцию по установке можно посмотреть в его репозитории
    2. Переместите папку titles из корня карты памяти в папку sxos
      • Саму папку, а не её содержимое
    3. Исправьте атрибуты по инструкции ниже
    4. enjoy

    ДЕЛАТЬ ОБЯЗАТЕЛЬНО после ручной установки

    1. Запустите приставку без карты памяти с помощью payload.bin
    2. Кнопками громкости и кнопкой включения перейдите в меню tools
    3. Вставьте карту памяти в приставку
    4. Выберите Unset archive bit и дождитесь окончания установки
    5. Выберите Back -> Launch - >YOUR_OS_NAME
    Обновление кефира прямо на приставке
    1. Запустите HBL
    2. Выберите Kefir Updater
      • Требуется подключение к интернету!
    3. Выберите ОС, которую используете
    4. По окончанию установки, перезагрузите приставку
